# SparRechner
Mit diesem Java Projekt versuche ich einen Ersparnis Rechner App zu verviklichen. Um dieser App zu benutzen soll man die monatliche Einzahlung, die jährlich Zinsenzahlung, und das anzahl der Monate eingeben. Nach der Berechnung bekommt man seinem Schlusssaldo. Außerdem man kann auch die montaliche Einzahlung berechnen wenn man die jährlich Zinsenzahlung, das anzahl der Monate und den Schlusssaldo eingeben.
